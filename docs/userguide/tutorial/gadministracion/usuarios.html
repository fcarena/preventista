Gestión de Usuarios

En esta sección se administran todo lo relacionado a los usuarios del sistema.

  • Listar todos los usuarios
  • Nuevo usuario
  • Modificar un usuario
  • Eliminar un usuario

Listar todos los usuarios

Para listar todos los usuarios del sistema se debe hacer click en la opción de menu 'Usuarios' del menú principal.

En esta sección se podrá filtrar lo usuarios existentes en el sistema por 'nombre de usuario', 'nombre' y 'apellido'.

La contraseña del usuario se visualiza encriptada.

Nuevo usuario

Para crear un nuevo usuario, se debe hacer clicl en el botón 'Nuevo' de la pantalla principal de usuarios. El sistema mostrara el siguiente formulario a ser completado por el formulario.

Los campos que contiene un '*' son abligatorios.

Modificar un usuario

Aqui podrias modifcar los datos de un usuario del sistema.

Los campos que contiene un '*' son abligatorios.

Eliminar un usuario

Si se hace click en el botón eliminar, el sistema pedirá confirmación para realizar tal acción.
